# Promise Landing Page

Sitio corporativo multipágina de **Promise** construido con **Vite + JavaScript Vanilla**.

## Requisitos

- Node.js 20+ (recomendado 22+)
- npm 10+

## Scripts

- `npm run dev`: entorno local
- `npm run build`: build de producción
- `npm run preview`: servir build localmente

## Variables de entorno

Crea un archivo `.env` opcional con:

```bash
VITE_ELEVENLABS_MODE=demo
VITE_ELEVENLABS_AGENT_ID=
```

- `VITE_ELEVENLABS_MODE`: `demo` o `live`
- `VITE_ELEVENLABS_AGENT_ID`: id público del agente de ElevenLabs para modo live

## Páginas

- `/index.html`: Inicio
- `/precios.html`: Precios
- `/quienes-somos.html`: Quiénes somos
- `/como-funciona.html`: Cómo funciona
